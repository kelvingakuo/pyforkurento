KMS WebRTC in loopback using pyforkurento

This web application shows a WebRtcEndpoint connected to itself (loopback).

To run this demo follow these steps:

  1. Click on Start to begin the stream negotiation.
  2. Grant access to the camera and microphone.
  3. Click on Stop to finish the test.
  4. Refresh the page to go again.
